# Lumeniax — Écosystème Digital Premium

Site vitrine premium pour **Lumeniax**, fusion de MSA Rising Digital et Lumenia.

## Sous-marques
- **Lumeniax Studio** — Agence de transformation digitale (`/studio/*`)
- **Lumeniax Academy** — Plateforme éditoriale & formation (`/academy/*`)

## Installation

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

Le build est généré dans `dist/` — déployable sur Vercel, Netlify ou tout hébergeur statique.

## Stack

React 19 · TypeScript · Vite 7 · Tailwind CSS v4 · Framer Motion · Wouter · Radix UI
