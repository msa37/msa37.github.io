DEMARRAGE RAPIDE (5 MINUTES)
=============================

ETAPE 1 : Creer un depot GitHub
--------------------------------
1. Allez sur https://github.com/new
2. Nommez le depot : rising-digital-ia
3. Choisissez "Public"
4. Cliquez sur "Create repository"

ETAPE 2 : Pousser le code sur GitHub
-------------------------------------
cd rising-digital-ia
git init
git add .
git commit -m "Initial commit: Rising Digital IA website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/rising-digital-ia.git
git push -u origin main

Remplacez YOUR_USERNAME par votre nom d'utilisateur GitHub.

ETAPE 3 : Configurer Netlify
-----------------------------
1. Allez sur https://netlify.com
2. Cliquez sur "Add new site" → "Import an existing project"
3. Selectionnez "GitHub" comme provider
4. Autorisez Netlify a acceder a vos depots
5. Selectionnez le depot "rising-digital-ia"
6. Configurez les parametres :
   - Build command : pnpm build
   - Publish directory : dist
7. Cliquez sur "Deploy site"

ETAPE 4 : Verifier le deploiement
---------------------------------
- Attendez que le build soit termine (environ 2-3 minutes)
- Cliquez sur le lien fourni par Netlify pour voir votre site en direct
- Verifiez que toutes les pages fonctionnent correctement

✅ C'est fait ! Votre site est maintenant en ligne !

Pour utiliser votre propre domaine (ex: risingdigitalai.com) :

APRES LE DEPLOIEMENT
====================

Mettre a jour le site :
  1. Faites des modifications locales
  2. Testez avec : pnpm dev
  3. Commitez : git add . && git commit -m "Description"
  4. Poussez : git push origin main
  5. Netlify deploiera automatiquement !

Surveiller les deploiements :
  - Sur Netlify, allez dans "Deploys"
  - Vous verrez l'historique de tous les deploiements
  - Cliquez sur un deploiement pour voir les details

Analyser les performances :
  - Sur Netlify, allez dans "Analytics"
  - Consultez les statistiques de visite
  - Identifiez les pages les plus populaires

DEPANNAGE
=========

Erreur : "pnpm not found"
  → Verifiez que pnpm-lock.yaml est dans le depot
  → Netlify doit detecter pnpm automatiquement

Erreur : "Build failed"
  → Verifiez les logs de build sur Netlify
  → Testez localement : pnpm build
  → Assurez-vous que toutes les dependances sont installees

Site affiche une page blanche
  → Verifiez la console du navigateur pour les erreurs
  → Assurez-vous que les chemins d'images sont corrects
  → Verifiez que dist est le bon repertoire de publication

Netlify ne detecte pas les changements
  → Attendez quelques secondes apres le push
  → Verifiez que le webhook GitHub est configure
  → Allez dans "Deploy settings" et cliquez sur "Trigger deploy"
